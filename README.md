# Analisador de Razão Contábil

## Descrição

Ferramenta profissional para análise e conferência de relatórios de razão contábil em formato PDF. Permite importar múltiplos arquivos PDF, extrair dados contábeis e visualizar informações detalhadas de forma organizada.

## Características

- ✅ **Funciona offline** - Nenhuma dependência de servidor
- ✅ **Múltiplos PDFs** - Processa vários arquivos simultaneamente
- ✅ **Interface responsiva** - Funciona em desktop, tablet e mobile
- ✅ **Sem instalação** - Basta abrir o arquivo no navegador
- ✅ **Seguro** - Todos os dados processados localmente no seu navegador

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Clique em "Selecionar Arquivos" ou arraste PDFs para a área designada
3. Os dados serão processados automaticamente
4. Navegue pelas abas para ver resultados e detalhes

## Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Nenhuma instalação necessária
